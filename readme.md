# 🎓 Certify – Certificate Management & Verification Plugin for WordPress

![WordPress Version](https://img.shields.io/badge/WordPress-4.0+-blue.svg)
![Tested Up To](https://img.shields.io/badge/Tested%20Up%20To-6.8.1-brightgreen.svg)
![PHP Version](https://img.shields.io/badge/PHP-5.4%2B-yellow.svg)
![License](https://img.shields.io/badge/License-GPLv2%20or%20later-blue.svg)

Certify is a simple yet powerful WordPress plugin built for educational institutions, training centers, and organizations to seamlessly manage and verify course certificates online.

---

## ✨ Features

- 🧾 Add, edit, and delete certificates via admin panel  
- 📥 Bulk upload certificates using CSV files  
- 🔍 Certificate verification via front-end search form  
- 🧑‍💻 Clean and intuitive UI for both admins and users  
- 🛡️ Lightweight, secure, and easy to integrate  

---

## 🚀 Installation

1. Download the Certify plugin ZIP file.
2. Go to **WordPress Admin → Plugins → Add New → Upload Plugin**.
3. Upload the ZIP file and click **Install Now**.
4. Activate the plugin.
5. Navigate to the **Certify** menu in your dashboard to begin managing certificates.
6. To add the verification form to any page or post, use the shortcode:

```shortcode
[get_certificate_search_form]
```
